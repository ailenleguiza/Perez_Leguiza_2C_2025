Proyecto Final EMG
Proyecto que integra comunicación Bluetooth Low Energy (BLE) con el procesamiento de señales EMG. Implementa la adquisición analógica, el cálculo de la FFT y la transmisión inalámbrica de los datos, permitiendo visualizar el espectro en una app móvil y evaluar la fatiga muscular mediante análisis espectral.

void read_data ( uint8_t * data,
uint8_t length )

Función de callback que se ejecuta cuando se reciben datos por BLE.

Si se recibe el carácter 'R', notifica la tarea EMGTask para procesar una nueva ventana de datos EMG.
Si se recibe el carácter 'B', se borra la gráfica de la FFT y se limpian los vectores de datos asociados.

Parameters
dataPuntero al arreglo de bytes recibidos.
lengthLongitud del mensaje recibido.
